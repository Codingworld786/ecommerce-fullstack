1. **Place your image files in this folder:**  
   `ecommerce/static/images/products/`

2. **Use these exact filenames** so they match each product:

   | Filename       | Product                    |
   |----------------|----------------------------|
   | `blazer.jpg`   | Classic Linen Blazer       |
   | `dress.jpg`    | Floral Midi Dress          |
   | `trousers.jpg` | High-Waist Wide Leg Trousers |
   | `sweater.jpg`  | Cashmere Crew Neck Sweater |
   | `croptop.jpg`  | Ribbed Crop Top            |
   | `coat.jpg`     | Wool Blend Coat            |
   | `shirt.jpg`    | Oxford Cotton Shirt        |
   | `chinos.jpg`   | Slim Fit Chinos            |
   | `jumper.jpg`   | Merino Wool Jumper         |
   | `denim.jpg`    | Denim Jacket               |
   | `joggers.jpg`  | Jogger Sweatpants          |
   | `belt.jpg`     | Leather Belt               |

3. **Formats:** `.jpg`, `.jpeg`, `.png`, or `.webp` all work. If you use a different extension (e.g. `blazer.png`), change the `"image"` value for that product in `app.py` to match (e.g. `"image": "blazer.png"`).

If a file is missing, the site shows a letter placeholder instead of the image.
